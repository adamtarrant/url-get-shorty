URL Shortener

Please use this API by submitting /create/:urltoshorten. To use shortened url simply append onto end of this url.

Create shortened URL - Example input

  • http://localhost:8080/create/www.google.com

Create shortened URL - Example output

  • {original_url: "www.google.com","short_url": 1}

Use shortened URL - Example input

  • http://localhost:8080/1

Use shortened URL - Example output

This will then redirect to www.google.com
